# Explore 출력 형식

## find-usage 출력

### `{대상}` 사용처

| # | 파일 | 라인 | 컨텍스트 |
|---|------|------|----------|
| 1 | `src/...` | L42 | 호출/import/참조 설명 |

**요약**: 총 N개 파일, M개 위치에서 사용

---

## find-deps 출력

### `{대상}` 의존성

**내부 의존성** (이 파일이 import하는 것):
| 모듈 | 용도 |
|------|------|
| `src.models.user` | User 모델 |

**역방향 의존성** (이 파일을 import하는 것):
| 모듈 | 용도 |
|------|------|
| `src.routers.auth` | 인증 처리 |

**외부 패키지**:
| 패키지 | 버전 | 용도 |
|--------|------|------|

**의존성 그래프**:
```
A -> B -> C
       -> D
```

---

## find-impact 출력

### `{대상}` 변경 영향

**직접 영향** (1-depth):
| 파일 | 영향 내용 |
|------|----------|

**간접 영향** (2-depth):
| 파일 | 경로 | 영향 내용 |
|------|------|----------|

**관련 테스트**:
| 테스트 파일 | 관련 테스트 수 |
|------------|--------------|

**위험도**: 🟢 낮음 / 🟡 중간 / 🔴 높음

---

## find-related 출력

### `{대상}` 관련 파일

**소스 파일**: 같은 모듈/디렉토리 파일
**테스트 파일**: 관련 테스트
**문서 파일**: 관련 문서

---

## analyze-structure 출력

### `{경로}` 구조 분석

**파일 통계**:
| 항목 | 값 |
|------|---|
| 파일 수 | N |
| 총 줄 수 | N |
| 평균 줄 수 | N |

**디렉토리 구조**:
```
path/
├── file1.py (N줄)
├── file2.py (N줄)
└── sub/
    └── file3.py (N줄)
```

**주요 클래스/함수**:
| 이름 | 파일 | 줄 수 | 설명 |
|------|------|-------|------|

---

## analyze-code 출력

### `{대상}` 코드 분석

**개요**: 파일/함수의 역할 한 줄 설명

**핵심 로직**:
1. 단계 1 설명
2. 단계 2 설명

**의존성**: 주요 import 목록
**특이사항**: 주의할 점

---

## find-duplicates 출력

### 중복 코드 탐지 결과

| # | 파일 A | 파일 B | 유사도 | 줄 수 |
|---|--------|--------|--------|-------|

**추출 권장**:
- 공통 로직 설명 -> 추출 위치 제안

---

## find-changes 출력

### 최근 변경 사항

| 파일 | 상태 | 변경 줄 수 | 주요 변경 |
|------|------|-----------|----------|

**요약**: N개 파일 변경, +A/-B 줄
