# 웹 조사

특정 주제에 대해 웹 검색을 수행하고 결과를 요약합니다.

## 실행할 작업

1. 검색 키워드 도출
2. WebSearch로 정보 수집
3. 관련 문서/블로그/공식 문서 우선 순위
4. 핵심 내용 요약

## 사용자 입력

$ARGUMENTS

- 주제: `"React best practices"`
- 기술: `"Python async/await patterns"`

## 주의사항

- 공식 문서 우선 참조
- 최신 정보 위주
- 복수 출처 교차 검증

## 출력 형식

.claude/templates/research/search-web.md 형식
