# 옵션 비교

두 가지 이상의 옵션을 비교 분석합니다.

## 실행할 작업

1. 비교 대상 파악
2. 비교 기준 설정
3. 각 옵션별 평가
4. 비교표 생성 및 추천

## 사용자 입력

$ARGUMENTS

- 비교: `"FastAPI vs Flask"`
- 선택: `"Redis vs Memcached"`
- 대안: `"REST vs GraphQL"`

## 주의사항

- 객관적 데이터 기반 (벤치마크, 통계)
- 프로젝트 맥락 고려
- 각 옵션의 최신 버전 기준

## 출력 형식

.claude/templates/research/compare.md 형식
