# 디자인 패턴/아키텍처 패턴 조사

특정 문제에 적합한 디자인 패턴이나 아키텍처 패턴을 조사합니다.

## 실행할 작업

1. 문제/요구사항 분석
2. 관련 패턴 검색
3. 패턴 설명 및 적용 사례
4. 장단점 및 적용 가이드

## 사용자 입력

$ARGUMENTS

- 문제: `"Observer pattern in Python"`
- 아키텍처: `"Microservices communication patterns"`

## 주의사항

- GoF 디자인 패턴 우선 참조
- 실제 코드 예시 포함

## 출력 형식

.claude/templates/research/search-patterns.md 형식
