# 태스크 분해

목표를 실행 가능한 작은 태스크로 분해합니다.

## 실행할 작업

1. 목표 분석 및 주요 단계 파악
2. 각 단계를 독립적인 태스크로 분해
3. 태스크 간 의존성 정의
4. 우선순위 설정
5. 실행 순서 결정

## 사용자 입력

$ARGUMENTS

- 목표 설명: "Add batch processing API"
- 옵션 `--from-spec <경로>`: 스펙 문서 기반 분해

## 분해 기준

- 각 태스크는 30분~2시간 내 완료 가능
- 명확한 입력/출력 정의
- 독립적으로 테스트 가능
- 의존성 최소화

## 출력 형식

.claude/templates/plan/breakdown.md 형식
