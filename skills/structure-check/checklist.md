# 품질 평가 기준

## 메트릭 임계값

| 메트릭 | 경고 🟡 | 심각 🔴 |
|--------|---------|---------|
| 파일 줄 수 | > 200 | > 300 |
| 순환 복잡도 | > 10 | > 15 |
| 함수 줄 수 | > 30 | > 50 |
| 파라미터 수 | > 3 | > 5 |
| 순환 의존성 | 감지 | - |
| 코드 중복도 | > 80% | - |

## 점검 커맨드

1. `explore:analyze-structure` - 파일 통계 및 구조
2. `explore:find-duplicates` - 중복 코드
3. `explore:find-deps` - 의존성 그래프

## 심각도 분류

### 🔴 Critical
- 순환 의존성
- 300줄 초과 파일
- 순환 복잡도 15 초과

### 🟡 Warning
- 200~300줄 파일
- 순환 복잡도 10~15
- 80% 이상 유사 코드

### 🔵 Info
- 빈 __init__.py
- 미사용 import
- 명명 규칙 불일치

## 권장 점검 주기

| 시점 | 범위 |
|------|------|
| 기능 추가 후 | 변경된 디렉토리 |
| PR 전 | 변경된 파일 전체 |
| 주간 | 전체 프로젝트 |
