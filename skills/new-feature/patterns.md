# 기능 유형별 패턴

## API Endpoint 추가

**탐색**: 기존 라우터 구조 파악
**계획**: 라우터 + 서비스 + 모델 설계
**구현**:
1. 데이터 모델 생성 (request/response)
2. 서비스 로직 구현
3. 라우터 엔드포인트 추가
4. 테스트 작성

## 서비스 레이어 추가

**탐색**: 기존 서비스 패턴 분석
**계획**: 인터페이스 + 의존성 설계
**구현**:
1. 서비스 인터페이스 정의
2. 구현체 작성
3. DI 등록 (필요시)
4. 테스트 작성

## UI 컴포넌트 추가

**탐색**: 기존 컴포넌트 패턴 분석
**계획**: 컴포넌트 계층 + 상태 관리 설계
**구현**:
1. 컴포넌트 파일 생성
2. 스타일 작성
3. 상태 관리 연결
4. 테스트 작성
